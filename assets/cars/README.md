# Car Images

Drop PNG files here named exactly the car id (lowercase, hyphens), e.g.:
- ferarry-v60.png
- mercedes-gls.png
- vw-golv.png

Recommended:
- Size: 400x260 (or any 1.5:1 ratio)
- Background: transparent or white
- Side-view product shot style

After adding a file, uncomment the matching line in 'src/content/carImages.ts'.
